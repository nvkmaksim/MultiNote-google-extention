# MultiNote-google-extention
Многофункциональное расширение для Chrome, которое позволяет сохранять текстовые заметки, скриншоты, голосовые сообщения и записи экрана в один клик.

Для записи голосовых сообщений возможно потребуется дать дополнительные разрешения на использование микрофона через настройки браузера
